# Pitch of My Personal Flavour: TabulaRasa

TabulaRasa is my experiment in building a modular life-simulation engine: a small digital world where agents can exist, move, make decisions, and interact with their environment. The project started from the idea of a "blank slate", a world that is simple at first, but structured well enough to grow into something more complex over time.

The current version focuses on the foundation of the simulation. Agents move through a grid-based world, react to needs such as hunger, find food, follow routes, and execute actions through a clear pipeline. Instead of making everything happen directly, the simulation separates intention, validation, movement, action execution, and world updates. This makes the project less about creating one quick prototype and more about designing a system that can scale.

My personal flavour in this project is the combination of technical structure and emergent behaviour. I am interested in how simple systems can produce behaviour that feels alive. A single agent choosing to wander or eat is simple, but the architecture is meant to support future layers such as jobs, routines, social relationships, memory, economy, and environmental feedback. I like projects where the code is not only functional, but also leaves room for imagination.

TabulaRasa is also an experiment in clean software architecture. The project is split into separate parts: the world stores what exists, agents decide what they want to do, the simulation contains the rules, and the web interface makes the world visible and editable. This separation helps me explore how larger simulations, games, or agent-based systems can be built without turning into chaos too quickly.
